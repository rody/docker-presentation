les composants de docker

  • Images (Build)
  • Registry (Ship)
  • Container (Run)

Image (Build)

  • Elle contient un programe et ses dépendances
  • Elle peut être publiée sur un 'registry'

Registry (Ship)

  • Stocke et distribue les images docker
  • un registre global est dispo à https://registry.hub.docker.com/
  • Possibilité d'avoir un registre privé (docker run registry)

Container (Run)

Il est crée à partir d'une image

Un conteneur execute le processus défini dans l'image

Il est isolé des autres conteneurs et de l'OS hôte

Il possède sa propre configuration réseau
